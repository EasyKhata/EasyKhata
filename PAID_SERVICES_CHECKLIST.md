# Cost & Security Precautions Checklist

## 1. Usage & Billing
- [ ] Set up billing alerts for Firebase and Vercel
- [ ] Review usage dashboards weekly
- [ ] Know free tier limits for each service

## 2. Optimize Usage
- [ ] Minimize Firebase reads/writes in code
- [ ] Compress images/assets for Vercel hosting
- [ ] Use caching and lazy loading

## 3. Security
- [ ] Use strict Firebase security rules
- [ ] Never expose API keys/secrets in frontend
- [ ] Enable 2FA for Google accounts

## 4. Email
- [ ] Use app passwords for SMTP if needed
- [ ] Avoid bulk/marketing emails from main account

## 5. Domain
- [ ] Enable auto-renewal for Squarespace domain
- [ ] Keep DNS records up to date

## 6. Backup
- [ ] Regularly export Firebase data
- [ ] Backup codebase (git, cloud)

---

> Review this checklist monthly and after any major changes or launches.
